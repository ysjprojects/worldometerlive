Introduction

Worldometer Live is an API service that scrapes live data from https://www.worldometers.info/ using Selenium.
Although Worldometer provides a trove of insightful data, they do not provide an API service for these data. Worldometer Live kicks in by allowing users to query a multitude of worldwide statistics in real time (with a 15 min margin of error). Users can choose to query specific fields, query by categories, or query everything.
